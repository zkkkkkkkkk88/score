# 体彩足球赛事雷达

一个中文足球赛事分析面板，用于展示中国竞彩网公开赛事、实时比分、赛前预测、购买方案、历史方案和命中复盘。

> 说明：本项目只做信息整理和本地模型分析，不展示赔率，不保证命中，不构成投注建议。赛事、竞彩编号、销售状态和比分来自中国竞彩网公开接口；购买方案和概率为本地模型估算。

## 功能

- 赛事总览：今日重点、每日方案汇总、明日预选池。
- 赛事分析：左侧赛事列表独立滚动，右侧分析固定显示；支持全部、今日、明日和可购买筛选。
- 单场玩法：胜平负、让球胜平负、具体总进球数、比分、半全场。
- 赛果对比：玩法卡片上方显示赛前预测，下方显示完赛真实结果，并自动标记正确或错误。
- 购买方案：按二串一、三串一、三串二、四串一、四串二分组；每组最多展示 5 个方案，适配每天大量串单。
- 历史方案：按已完赛和未完赛左右分栏，再按日期和串单类型归类。
- 命中复盘：按日期折叠归档，每一天下面可以点开查看具体购买方案和每场结果。
- 自动刷新：实时服务默认每 60 秒刷新一次 Sporttery 数据，并重新生成购买方案。

## 本地运行

推荐使用实时服务启动：

```powershell
node scripts/serve-live.js
```

访问：

```text
http://127.0.0.1:4173
```

实时服务会：

- 启动时拉取一次中国竞彩网公开数据。
- 默认每 60 秒刷新一次。
- 提供 `/api/matches` 给前端读取最新数据。
- 数据过期时自动运行 `scripts/update-real-data.js`。

## 手动刷新数据

```powershell
node scripts/update-real-data.js
```

可选环境变量：

```powershell
$env:SPORTTERY_PAGE_SIZE="80"
$env:SCORE_UPDATE_INTERVAL_MS="60000"
$env:PORT="4173"
node scripts/serve-live.js
```

## 静态打开

可以用普通静态服务查看页面：

```powershell
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

但普通静态服务不会自动刷新真实数据。需要实时更新时，请使用：

```powershell
node scripts/serve-live.js
```

## 测试

```powershell
node tests/dashboard-smoke.test.js
```

该测试会检查页面基础渲染、玩法标签、购买方案、历史方案归类、命中复盘折叠、比分与总进球一致性等关键行为。

## 数据与复盘规则

- 当前赛事数据写入 `data/matches.json`。
- 每次生成的购买方案会保存到 `planArchive`。
- 完赛后系统会自动判断每个方案和每个单场选择是否正确。
- 未完赛场次保持待复盘状态。
- 历史命中率只统计项目实际保存过的购买方案，不补造接入前的历史结果。
